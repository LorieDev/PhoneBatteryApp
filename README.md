# PhoneBatteryApp

## Overview

PhoneBatteryApp is a beginner DevOps and Cloud project built with Java.
The application simulates a phone battery system where users can update the battery level and modify the phone brand.

This project was created to practice real-world DevOps tools and cloud workflow integration.

---

## Technologies Used

* Java
* Maven
* JUnit 5
* Git & GitHub
* Jenkins
* Docker
* AWS EC2
* Terraform
* Kubernetes (Minikube)
* Logging with kubectl logs
* Apache JMeter

---

## Features

* Create Phone objects
* Charge phone battery
* Update phone brand
* Unit testing with JUnit
* Maven build automation
* Jenkins CI integration
* Docker containerization
* AWS cloud infrastructure
* Terraform Infrastructure as Code
* Kubernetes deployment
* Kubernetes logging
* JMeter performance testing setup
